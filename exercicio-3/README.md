# Exercício 3: Cálculo de Fatorial

## Descrição
Implementação de uma função em Python que calcula o fatorial de um número inteiro não negativo (`n!`), conforme definição matemática:
- `n! = n × (n-1) × ... × 1`  
- `0! = 1` (caso base).

**Exemplos do enunciado**:  
- `5! = 120`  
- `0! = 1`  
- `3! = 6`
